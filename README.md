## :star: Get Started
``` bash
# clone
git clone https://github.com/NguyenHoangTrungII/api-linhkien-donet.git
```

``` bash
# install dependencies
npm install
```
Or use `yarn`
``` bash
# install dependencies by yarn
yarn install
```

``` bash
# run project
npm start
```
